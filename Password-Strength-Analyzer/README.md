# Password Strength Analyzer

## Overview

Password Strength Analyzer is a Python-based cybersecurity project that evaluates password security. The tool checks password complexity, detects common passwords, calculates entropy, and provides recommendations for creating stronger passwords.

## Features

* Password strength evaluation (Weak, Medium, Strong)
* Common password detection
* Password entropy calculation
* Security recommendations
* Command-line interface

## Technologies Used

* Python
* Regular Expressions (Regex)
* Cybersecurity Fundamentals
* Secure Coding Practices

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run the following command:

python password_analyzer.py
