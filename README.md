# Password Strength Analyzer

## Overview

Password Strength Analyzer is a web-based cybersecurity project built using Flask, HTML, CSS, and Python. It helps users evaluate the strength of their passwords based on multiple security factors.

## Features

* Password strength evaluation
* Entropy calculation
* Detection of uppercase, lowercase, numbers, and special characters
* Security score generation
* Password improvement suggestions
* User-friendly web interface

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript

## How to Run

1. Clone the repository:
   git clone https://github.com/your-username/Password-Strength-Analyzer.git

2. Install dependencies:
   pip install flask

3. Run the application:
   python app.py

4. Open:
   http://127.0.0.1:5000
